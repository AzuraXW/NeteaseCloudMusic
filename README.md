# 云音乐

## 项目概述

本项目是模仿网易云客户端而成的，仅作学习交流用途。本项目中不包含任何的支付接口，如侵权请Issues

## 本地预览

``````
npm install     #下载依赖
npm run serve   #开启本地服务器
``````

## 构建

```
npm run build
```

## Eslint

```
npm run lint
```

## 完成的功能

- **登录**
- **注册**
- **发现音乐**
  - **个性推荐**
  - **歌单**
- **用户歌单**
  - **用户添加新歌单**
  - **用户删除歌单**
- 网友精选歌单
- 精品歌单排行榜
- 播放功能
- 歌单展示
- 心动模式
